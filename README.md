# TypeScript 101: Explicit Types

- Explicit types
```
let character:string;
let age:number;
let isLoggedIn:boolean;
```
- Arrays
```
let names:string[]
```

- Union types
```
let id:string|number
let mixed:(string|number)[]
```

- Objects
```
let obj1:object;
let obj2:{
    name:string,
    age:number,
    city:string
}
```
